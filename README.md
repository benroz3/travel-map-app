Travel Map fullstack app using MERN & Leaflet map library.

backend:

1. create .env file using the example
2. Open first terminal
3. type: "cd backend"
4. type: "npm i"
5. type: "npm start"

frontend:

1. create .env file using the example
2. Open second terminal
3. type: "cd frontend"
4. type: "npm i"
5. type: "npm start"
